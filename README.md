# 📊 Business Dashboard

A Full Stack Business Dashboard built using React.js, FastAPI, and MySQL.

---

## 🚀 Features

- City-wise Business Count
- Category-wise Business Count
- Source-wise Business Count
- Bar Charts
- Pie Charts
- Dark / Light Mode
- Responsive Dashboard UI
- MySQL Database Integration
- FastAPI REST APIs

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- Recharts

### Backend
- FastAPI
- Python

### Database
- MySQL

---

## 📈 Visualizations

- Bar Charts
- Pie Charts

---

## ⚡ How to Run

### Backend

```bash
cd backend
venv\Scripts\activate
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## 🌐 API Endpoints

- `/city-wise`
- `/category-wise`
- `/source-wise`
